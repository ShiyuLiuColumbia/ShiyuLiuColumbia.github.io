232. Implement Queue using Stacks

Queue是先进先出，Stack是先进后出，我们需要用两个Stack来实现一个Queue。

  1. push的时候不动，pop的时候通过一个额外的临时Stack来改变顺序。
    push的时间复杂度永远是O(1)，pop的时间复杂度不一定，当临时Stack还有值得时候是O(1)，没有值的时候需要O(n)。

  2. 在push的时候就改变好顺序，pop的时候直接取出就好。
    push：O(n)，pop：O(1)
