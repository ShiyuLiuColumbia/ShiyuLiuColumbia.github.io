222. Count Complete Tree Nodes

最直观的做法肯定是直接进行递归，时间复杂度O(n)。

但是我们发现这道题一个关键点是：树是一个完美树(perfect tree)
所以正确的做法是运用树的高度与node个数的关系：如果一个树是完全的完美树(在完美树的基础上最后一层也铺满)，那么n=2^h-1
